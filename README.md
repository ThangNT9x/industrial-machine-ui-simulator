# Industrial Machine UI Simulator

## Overview
A C#/.NET desktop application that simulates an industrial machine operator UI.

This project is inspired by real industrial machine software and focuses on:
- machine state flow
- operator dashboard UI
- event/alarm logging
- JSON configuration
- SQLite local storage
- mock PLC / mock communication

## Goals
- Practice industrial software architecture
- Build a portfolio-ready desktop application
- Learn machine state management and clean project structure
